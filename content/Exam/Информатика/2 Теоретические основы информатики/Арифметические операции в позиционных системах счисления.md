Арифметические операции в позиционных системах счисления включают сложение, вычитание, умножение и деление. Хотя принципы выполнения операций схожи для разных систем счисления, существуют особенности, зависящие от основания системы. Рассмотрим выполнение арифметических операций на примерах для двоичной, восьмеричной и шестнадцатеричной систем.

### Сложение

При сложении чисел в позиционных системах применяются те же правила переноса, что и в десятичной системе, но с учетом основания [[системы счисления]].

#### Пример: Сложение в двоичной системе

Пусть $a = 1011_2$ и $b = 0110_2$. 

```
  1011
+ 0110
-------
  10001
```

#### Пример: Сложение в восьмеричной системе

Пусть $a = 57_8$ и $b = 23_8$. 

```
   57
+  23
-------
   80
```

#### Пример: Сложение в шестнадцатеричной системе

Пусть $a = A1_{16}$ и $b = 2C_{16}$. 

```
   A1
+  2C
-------
   CC
```

### Вычитание

При вычитании чисел в позиционных системах могут потребоваться заимствования из старших разрядов, если разность оказывается меньше нуля.

#### Пример: Вычитание в двоичной системе

Пусть $a = 1011_2$ и $b = 0110_2$. 

```
  1011
- 0110
-------
   101
```

#### Пример: Вычитание в восьмеричной системе

Пусть $a = 75_8$ и $b = 31_8$. 

```
   75
-  31
-------
   44
```

#### Пример: Вычитание в шестнадцатеричной системе

Пусть $a = FF_{16}$ и $b = A1_{16}$. 

```
   FF
-  A1
-------
   EE
```

### Умножение

Умножение выполняется путем перемножения каждой цифры одного множителя на каждую цифру другого множителя и суммирования результатов с учетом переносов.

#### Пример: Умножение в двоичной системе

Пусть $a = 101_2$ и $b = 11_2$. 

```
       101
*        11
-----------
       101
+     1010
-------------
      1111
```

#### Пример: Умножение в восьмеричной системе

Пусть $a = 37_8$ и $b = 17_8$. 

```
      37
*     17
----------
      65
    296
+   444
-----------
    605
```

#### Пример: Умножение в шестнадцатеричной системе

Пусть $a = AB_{16}$ и $b = CD_{16}$. 

```
      AB
*     CD
-----------
      DB00
    AC00
+   A0000
-----------
    ADBCD
```

### Деление

Деление в позиционных системах выполняется аналогично десятичному делению, но с учетом оснований систем.

#### Пример: Деление в двоичной системе

Пусть $a = 1101_2$ и $b = 11_2$. 

```
     1101 / 11 = 100_2
```

#### Пример: Деление в восьмеричной системе

Пусть $a = 75_8$ и $b = 5_8$. 

```
     75 / 5 = 15_8
```

#### Пример: Деление в шестнадцатеричной системе

Пусть $a = FF_{16}$ и $b = A_{16}$. 

```
     FF / A = 19_16
```

### Особенности

- **Перенос**: Когда результат сложения двух цифр превышает основание системы, происходит перенос единицы в старший разряд.
- **Заимствование**: При вычитании, если уменьшаемое меньше вычитаемого, производится заимствование из старшего разряда.
- **Нули и ведущие нули**: Нули в конце числа не влияют на его значение, но ведущие нули перед числом игнорируются.
