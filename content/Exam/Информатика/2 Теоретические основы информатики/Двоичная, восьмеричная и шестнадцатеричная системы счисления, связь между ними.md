Двоичная, восьмеричная и шестнадцатеричная [[системы счисления]] являются важными представителями позиционных систем счисления, используемых в вычислительной технике и программировании. Эти системы тесно связаны друг с другом благодаря своим основаниям, что упрощает перевод чисел между ними.

### Основные понятия

- **Основание [[системы счисления]]**: это количество уникальных символов (цифр), используемых для представления чисел. Основанием двоичной системы является 2, восьмеричной — 8, а шестнадцатеричной — 16.

### Связь между системами

#### Двоичная и восьмеричная системы

Восьмеричная система удобна для работы с двоичными числами, так как каждые три бита (двоичных цифры) можно легко перевести в одну восьмеричную цифру. Это связано с тем, что $2^3 = 8$. Таблица соответствия двоичных и восьмеричных цифр:

| Двоичное | Восьмеричное |
|----------|---------------|
| 000      | 0             |
| 001      | 1             |
| 010      | 2             |
| 011      | 3             |
| 100      | 4             |
| 101      | 5             |
| 110      | 6             |
| 111      | 7             |

**Пример перевода из двоичной в восьмеричную систему:**

Число $10111001_2$ разбиваем на группы по три бита справа налево: $101\,110\,001$. Теперь переводим каждую группу в восьмеричную цифру: $101 = 5$, $110 = 6$, $001 = 1$. Получается число $561_8$.

#### Двоичная и шестнадцатеричная системы

Шестнадцатеричная система еще удобнее для работы с большими двоичными числами, так как каждые четыре бита соответствуют одной шестнадцатеричной цифре. Это связано с тем, что $2^4 = 16$. В шестнадцатеричной системе используют цифры от 0 до 9 и буквы A-F для обозначения цифр от 10 до 15. Таблица соответствия двоичных и шестнадцатеричных цифр:

| Двоичное     | Шестнадцатеричное |
|--------------|--------------------|
| 0000         | 0                  |
| 0001         | 1                  |
| 0010         | 2                  |
| 0011         | 3                  |
| 0100         | 4                  |
| 0101         | 5                  |
| 0110         | 6                  |
| 0111         | 7                  |
| 1000         | 8                  |
| 1001         | 9                  |
| 1010         | A                  |
| 1011         | B                  |
| 1100         | C                  |
| 1101         | D                  |
| 1110         | E                  |
| 1111         | F                  |

**Пример перевода из двоичной в шестнадцатеричную систему:**

Число $10111001_2$ разбиваем на группы по четыре бита справа налево: $1011\,1001$. Теперь переводим каждую группу в шестнадцатеричную цифру: $1011 = B$, $1001 = 9$. Получается число $B9_{16}$.

### Преимущества и применение

- **Двоичная система** используется непосредственно в цифровых устройствах, так как основана на двух состояниях (включено/выключено).
- **Восьмеричная и шестнадцатеричная системы** позволяют компактнее представлять большие двоичные числа, что удобно для программистов и инженеров.
