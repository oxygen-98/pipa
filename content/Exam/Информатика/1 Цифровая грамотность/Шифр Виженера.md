Шифр Виженера — это многоалфавитный шифр замены, который был разработан французским дипломатом Блезом де Виженером в XVI веке. Этот шифр является усовершенствованием классического шифра Цезаря и предназначен для повышения стойкости к криптоанализу.

### Принцип работы шифра Виженера

Шифр Виженера использует таблицу, называемую квадратом Виженера, для шифрования текста. Таблица представляет собой матрицу размером $26 \times 26$, где строки соответствуют алфавиту, а столбцы — возможным сдвигам.

### Шаг шифрования

1. Выбирается ключевое слово или фраза, которая определяет последовательность сдвигов для каждой буквы текста.
2. Буквы ключевого слова используются для выбора строки в таблице Виженера, начиная с первой буквы ключевого слова.
3. Буквы открытого текста заменяются соответствующими буквами из выбранной строки.

### Пример шифрования

Пусть у нас есть открытый текст "ATTACKATDAWN" и ключевое слово "LEMON". Тогда процесс шифрования будет следующим:

1. Запишем открытый текст и повторим ключевое слово столько раз, сколько нужно:

```
   ATTACKATDAWN
   LEMONLEMONLE
```

1. Используя квадрат Виженера, найдем соответствие каждой пары букв:

```
   A (открытый текст) + L (ключ) = X (шифротекст)
   T (открытый текст) + E (ключ) = X (шифротекст)
   T (открытый текст) + M (ключ) = K (шифротекст)
   ...
```

Получившийся шифротекст: "XXWKVIMYWMND".

### Достоинства и недостатки шифра Виженера

#### Достоинства:
- Повышенная стойкость к частотно-статистическому анализу по сравнению с одноалфавитными шифрами, такими как [[шифр Цезаря]].
- Относительно простая реализация.

#### Недостатки:
- Неустойчив к современным методам криптоанализа, таким как атака на основе известного открытого текста.
- Требуется надежный механизм распределения ключей.
