Шифры простой замены — это класс криптографических методов, в которых каждый символ открытого текста заменяется на другой символ согласно заранее установленному правилу. Эти методы являются одними из самых простых и древних способов шифрования, однако они также считаются уязвимыми для криптоанализа.

### Механизм работы шифра простой замены

Основной идеей шифра простой замены является замена каждого символа исходного текста на определенный символ шифротекста. Замена производится по фиксированному правилу, известному обеим сторонам — отправителю и получателю. Например, если правило замены следующее:

```
A → Z
B → Y
C → X
...
Z → A
```

Тогда слово "HELLO" после шифрования превратится в "URYYB".

### Виды шифров простой замены

1. **[[Шифр Цезаря]]**:
   - Самый известный и простой шифр простой замены.
   - Каждой букве алфавита соответствует другая буква, смещённая на определенное число позиций вперёд. Например, смещение на 3 позиции даст следующий шифр:

```
A → D
B → E
C → F
...
X → A
Y → B
Z → C
```

1. **Шифр Атбаш**:
   - Алфавит переворачивается наоборот. Первая буква заменяется последней, вторая — предпоследней и так далее. Например:

```
A → Z
B → Y
C → X
...
M → N
N → M
O → L
...
Z → A
```

1. **Многоалфавитные шифры**:
   - Многоалфавитные шифры усложняют простую замену, используя несколько таблиц замены. Например, Vigenère Cipher, где каждый символ шифруется с использованием разных смещений.

### Недостатки шифров простой замены

1. **Частоты символов**:
   - В любом языке символы появляются с разной частотой. Например, в английском языке буква "E" встречается чаще остальных. [[Анализ]] частот появления символов в шифротексте может позволить взломщику определить правила замены.

2. **Маленький ключ**:
   - Простота замены делает ключи небольшими и предсказуемыми. В шифрах вроде Цезаря ключ — это просто число смещения, что делает их крайне уязвимыми.

3. **Легкость декодирования**:
   - Даже без знаний о ключе криптоанализатор может применить статистические методы, чтобы восстановить исходный текст.

### Историческое значение

Несмотря на слабость, шифры простой замены были популярны в древности и средневековье, так как их было легко применять и понимать. Они использовались в военных и дипломатических целях вплоть до эпохи Возрождения, когда появились более сложные методы шифрования.

### Современное применение

Сегодня шифры простой замены практически не применяются в реальных криптографических системах из-за их слабости. Тем не менее, они остаются важными в образовательных целях, помогая понять основы криптографии и методы криптоанализа.
