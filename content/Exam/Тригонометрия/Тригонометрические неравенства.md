Тригонометрические неравенства — это неравенства, включающие тригонометрические функции переменной, такие как синус, косинус, тангенс и др. Их решение требует понимания свойств тригонометрических функций и умения работать с областями их значений.

### Методы решения тригонометрических неравенств

1. **[[Анализ]] области значений функции**
   - Определение областей, где функция принимает положительные или отрицательные значения.
   - Проверка границ изменения функции.

2. **Преобразование неравенства**
   - Приведение неравенства к виду, удобному для анализа.
   - Использование тригонометрических тождеств.

3. **Использование тригонометрического круга**
   - Нахождение решений на единичной окружности.
   - Работа с периодичностью тригонометрических функций.

4. **Интервалов метод**
   - Разделение области определения функции на интервалы.
   - Исследование знаков функции в каждом интервале.

5. **Графический метод**
   - Построение графиков функций.
   - Сравнение графиков для нахождения областей выполнения неравенства.

### Примеры решения тригонометрических неравенств

Рассмотрим несколько примеров решения тригонометрических неравенств различными методами.

#### Пример 1: Простое неравенство

Решить неравенство $\sin x > \frac{1}{2}$.

Решение:
Используем свойство периодичности синуса. Знаем, что $\sin x > \frac{1}{2}$ при $x \in (\frac{\pi}{6} + 2\pi n, \frac{5\pi}{6} + 2\pi n)$, где $n$ — целое число.

#### Пример 2: Неравенство с заменой переменной

Решить неравенство $2\cos^2 x - 3\sin x > 0$.

Решение:
Пусть $u = \sin x$. Тогда неравенство примет вид $2(1-u^2) - 3u > 0$, или $2u^2 + 3u - 2 < 0$. Решая квадратное неравенство, получаем $u \in (-2, \frac{1}{2})$. Следовательно, $\sin x \in (-2, \frac{1}{2})$, откуда $x \in [\frac{7\pi}{6} + 2\pi n, \frac{11\pi}{6} + 2\pi n]$, где $n$ — целое число.

#### Пример 3: Неравенство с использованием тригонометрического круга

Решить неравенство $\tan x > 1$.

Решение:
Используя тригонометрический круг, находим, что $\tan x > 1$ при $x \in (\frac{\pi}{4} + \pi n, \frac{5\pi}{4} + \pi n)$, где $n$ — целое число.
