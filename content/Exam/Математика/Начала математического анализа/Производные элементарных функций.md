Производные элементарных функций — это набор основных формул, используемых для нахождения производных базовых математических функций. Эти формулы важны для понимания основ дифференциального исчисления и решения широкого круга задач. Давайте рассмотрим основные производные элементарных функций.

### 1. Степенная функция

Производная степенной функции $f(x) = x^n$ (где $n$ — любое вещественное число) определяется как:

$$f'(x) = nx^{n-1}.$$

### 2. [[Экспоненциальная функция]]

Производная экспоненты $f(x) = e^x$ (где $e$ — основание натуральных логарифмов) равна самой функции:

$$f'(x) = e^x.$$

### 3. Логарифмическая функция

Производная натурального логарифма $f(x) = \ln x$ (для $x > 0$) определяется как:

$$f'(x) = \frac{1}{x}.$$

### 4. Тригонометрические функции

Производные тригонометрических функций определяются следующим образом:

- Синус: $f(x) = \sin x$ → $f'(x) = \cos x$.
- Косинус: $f(x) = \cos x$ → $f'(x) = -\sin x$.
- Тангенс: $f(x) = \tan x$ → $f'(x) = \sec^2 x = \frac{1}{\cos^2 x}$.
- Котангенс: $f(x) = \cot x$ → $f'(x) = -\csc^2 x = -\frac{1}{\sin^2 x}$.
- Секанс: $f(x) = \sec x$ → $f'(x) = \sec x \tan x$.
- Косеканс: $f(x) = \csc x$ → $f'(x) = -\csc x \cot x$.

### 5. Обратные тригонометрические функции

Производные обратных тригонометрических функций:

- Арксинус: $f(x) = \arcsin x$ → $f'(x) = \frac{1}{\sqrt{1-x^2}}$ (при $-1 < x < 1$).
- Арккосинус: $f(x) = \arccos x$ → $f'(x) = -\frac{1}{\sqrt{1-x^2}}$ (при $-1 < x < 1$).
- Арктангенс: $f(x) = \arctan x$ → $f'(x) = \frac{1}{1+x^2}$.
- Арккотангенс: $f(x) = \operatorname{arcctg} x$ → $f'(x) = -\frac{1}{1+x^2}$.
- Арксеканс: $f(x) = \operatorname{arcsec} x$ → $f'(x) = \frac{1}{|x|\sqrt{x^2-1}}$ (при $|x| > 1$).
- Арккосеканс: $f(x) = \operatorname{arccsc} x$ → $f'(x) = -\frac{1}{|x|\sqrt{x^2-1}}$ (при $|x| > 1$).

### 6. Гиперболические функции

Производные гиперболических функций аналогичны тригонометрическим:

- Гиперболический синус: $f(x) = \sinh x$ → $f'(x) = \cosh x$.
- Гиперболический косинус: $f(x) = \cosh x$ → $f'(x) = \sinh x$.
- Гиперболический тангенс: $f(x) = \tanh x$ → $f'(x) = \sech^2 x = \frac{1}{\cosh^2 x}$.
- Гиперболический котангенс: $f(x) = \coth x$ → $f'(x) = -\csch^2 x = -\frac{1}{\sinh^2 x}$.
- Гиперболический секанс: $f(x) = \operatorname{sech} x$ → $f'(x) = -\operatorname{sech} x \tanh x$.
- Гиперболический косеканс: $f(x) = \operatorname{csch} x$ → $f'(x) = -\operatorname{csch} x \coth x$.
