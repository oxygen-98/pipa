### Наибольшее и наименьшее значение функции на промежутке

#### Определение

Наибольшее значение функции $f(x)$ на промежутке $[a, b]$ — это максимальное значение, которое функция принимает на этом промежутке. Наименьшее значение функции — это минимальное значение, которое функция принимает на этом промежутке.

#### Нахождение наибольшего и наименьшего значения

Для нахождения наибольшего и наименьшего значения функции $f(x)$ на замкнутом промежутке $[a, b]$ можно использовать следующую процедуру:

1. Найдите критические точки функции внутри промежутка, решив уравнение $f'(x) = 0$.
2. Вычислите значения функции в критических точках и на концах промежутка: $f(a)$ и $f(b)$.
3. Сравните полученные значения и выберите наибольшее и наименьшее.

#### Примеры

1. **Функция $f(x) = x^2$ на промежутке $[-1, 2]$**:
   - Критические точки: Нет (производная $f'(x) = 2x$ не равна нулю на $[-1, 2]$).
   - Значения на концах промежутка: $f(-1) = 1$, $f(2) = 4$.
   - Наибольшее значение: $f(2) = 4$.
   - Наименьшее значение: $f(-1) = 1$.

2. **Функция $f(x) = x^3 - 3x$ на промежутке $[-2, 2]$**:
   - Первая производная: $f'(x) = 3x^2 - 3$.
   - Критические точки: $x_1 = -1$, $x_2 = 1$.
   - Значения функции: $f(-2) = -10$, $f(-1) = 2$, $f(1) = -2$, $f(2) = 2$.
   - Наибольшее значение: $f(-1) = 2$.
   - Наименьшее значение: $f(-2) = -10$.
