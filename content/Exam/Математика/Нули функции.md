### Нули функции

#### Определение

Нулём функции $f(x)$ называется такое значение $x$, при котором функция обращается в ноль, то есть $f(x) = 0$.

#### Нахождение нулей функции

Чтобы найти нули функции, нужно решить уравнение $f(x) = 0$. Решение этого уравнения даст нам значения $x$, при которых функция равна нулю.

#### Примеры

1. **[[Линейная функция]] $f(x) = ax + b$**:
   - Нуль функции находится из уравнения $ax + b = 0$, откуда $x = -\frac{b}{a}$.

2. **Квадратичная функция $f(x) = ax^2 + bx + c$**:
   - Нули функции находятся из уравнения $ax^2 + bx + c = 0$. Решениями будут корни квадратного уравнения, найденные по формуле $x_{1,2} = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$.

3. **Тригонометрическая функция $f(x) = \sin x$**:
   - Нули функции находятся из уравнения $\sin x = 0$. Решениями будут $x = k\pi$, где $k$ — целое число.

4. **Степенная функция $f(x) = x^n - a$**:
   - Нуль функции находится из уравнения $x^n - a = 0$, откуда $x = \sqrt[n]{a}$.

#### Заключение

Нули функции играют важную роль в анализе её поведения и свойств. Они определяют точки, в которых функция пересекает ось $x$, и помогают понять характер изменений функции вблизи этих точек.
