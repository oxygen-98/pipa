Множество — это совокупность объектов, называемых элементами множества. Операции над множествами позволяют объединять, пересечение, дополнение и разность множеств, что является основой теории множеств.

### Основные операции над множествами

1. **Объединение множеств**:
   Объединение двух множеств $A$ и $B$ — это новое множество, содержащее элементы, принадлежащие хотя бы одному из исходных множеств. Записывается как $A \cup B$.

2. **Пересечение множеств**:
   Пересечение двух множеств $A$ и $B$ — это новое множество, содержащее элементы, общие для обоих исходных множеств. Записывается как $A \cap B$.

3. **Разность множеств**:
   Разность множеств $A$ и $B$ — это новое множество, содержащее элементы множества $A$, которые не принадлежат множеству $B$. Записывается как $A \setminus B$ или $A - B$.

4. **Дополнение множества**:
   Дополнение множества $A$ — это новое множество, содержащее все элементы универсального множества, которые не принадлежат множеству $A$. Записывается как $U \setminus A$ или $A^c$.

### Свойства операций над множествами

Операции над множествами обладают рядом свойств, которые упрощают работу с ними:

1. **Коммутативность**:
   - $A \cup B = B \cup A$
   - $A \cap B = B \cap A$

2. **Ассоциативность**:
   - $(A \cup B) \cup C = A \cup (B \cup C)$
   - $(A \cap B) \cap C = A \cap (B \cap C)$

3. **Идемпотентность**:
   - $A \cup A = A$
   - $A \cap A = A$

4. **Законы де Моргана**:
   - $(A \cup B)^c = A^c \cap B^c$
   - $(A \cap B)^c = A^c \cup B^c$

### Примеры

1. **Объединение множеств**:
   Пусть $A = \{1, 2, 3\}$ и $B = \{2, 3, 4\}$. Тогда объединение этих множеств:

$$A \cup B = \{1, 2, 3\} \cup \{2, 3, 4\} = \{1, 2, 3, 4\}.$$

1. **Пересечение множеств**:
   Используя те же множества $A$ и $B$, их пересечение:

$$A \cap B = \{1, 2, 3\} \cap \{2, 3, 4\} = \{2, 3\}.$$

1. **Разность множеств**:
   Разность множеств $A$ и $B$:

$$A \setminus B = \{1, 2, 3\} \setminus \{2, 3, 4\} = \{1\}.$$

1. **Дополнение множества**:
   Предположим, что универсальное множество $U = \{1, 2, 3, 4, 5\}$. Тогда дополнение множества $A$:

$$A^c = U \setminus A = \{1, 2, 3, 4, 5\} \setminus \{1, 2, 3\} = \{4, 5\}.$$
